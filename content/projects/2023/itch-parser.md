This project is a collection of tools for parsing the NASDAQ ITCH (Integrated Trade and Clearing Hub) protocol. It includes a parser for the protocol messages and a UDP server for handling binary data streams.

The components of this project are spread across a few repositories:
- [itch-parser-eric](https://github.com/EricSpencer00/itch-parser-eric)
- [UDP-server-binary](https://github.com/EricSpencer00/UDP-server-binary)
- [itch-parser](https://github.com/EricSpencer00/itch-parser)
