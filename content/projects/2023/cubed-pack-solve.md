A solver for packing 54 T-tetracubes into a 6x6x6 cube using Knuth's Dancing Links algorithm (Algorithm X). Finds all distinct tilings up to rotational symmetry and includes a Three.js 3D visualization web viewer. Built with Python.

The source code is available on GitHub: [https://github.com/EricSpencer00/cubed-pack-solve](https://github.com/EricSpencer00/cubed-pack-solve)
